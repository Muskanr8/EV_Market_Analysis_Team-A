# EV_Market_Analysis_Team-A

## Overview
This project uses machine learning (KMeans) to segment the Indian EV market based on:
- Vehicle_Sales
- Petrol/Diesel_Sales
- EV_Sales

The goal is to understand EV adoption levels and identify highâ€‘potential markets.

## Steps
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- Elbow Method & Silhouette Score
- KMeans clustering (k = 6)
- PCA visualization
- Cluster insights & recommendations

## Libraries Used
pandas, numpy, matplotlib, seaborn, scikit-learn

## Results
The analysis identified **6 market clusters**, showing:
- EV-ready regions  
- Transitional markets  
- ICE-dominant regions  
- Low-readiness areas
